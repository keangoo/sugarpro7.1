ForecastsApi - user

Summary:

This endpoint is used to return a user's id, user_name, full_name, first_name, last_name, and isManager param given a user's id.

Query Parameters:

Param Description Optional
user_id Returns user data for this specific user id  

Input Example:

{ user_id:seed_sally_id }

Output Example:

{ first_name: "Sally" full_name: "Sally Bronsen" id: "seed_sally_id" isManager: false last_name: "Bronsen" user_name: "sally" }
