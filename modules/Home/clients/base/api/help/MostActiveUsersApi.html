MostActiveUsers

Summary:

This endpoint is used to most active users for meetings, 'calls, inbound emails, and outbound emails.

Query Parameters:

Param Description Optional
days Returns most active users for last n days yes

Input Example:

{ days:30 }

Output Example:

{ "meetings": { "user_id": "seed_sally_id", "count": "17", "first_name": "Sally", "last_name": "Bronsen" }, "calls": {}, "inbound_emails": {}, "outbound_emails": { "user_id": "seed_sally_id", "count": "48", "first_name": "Sally", "last_name": "Bronsen" } }
