Overview

Updates a target list with selected records.

Summary

This endpoint is used mainly against the Users Contacts and Leads modules.

Query String Parameters

This endpoint does not accept any query string parameters.

Input Parameters

Name Type Description Required
module String The module of the ids being associated. True
recordIds Array The list of record ids to associate. True
prospectListId String The prospect list id. True

Input Example

{
   "prospectListId":"8f8e57d1-6ac2-889b-778e-513009728f5b",
   "module":"Contacts",
   "recordIds":[
      "ef1b40aa-5815-4f8d-e909-512d09617ac8"
   ]
}

Result

Name Type Description
<associated record id> Boolean A list of record ids and the association result.

Output Example

{
   "ef1b40aa-5815-4f8d-e909-512d09617ac8":true
}
